CSS Colors:

  • The color property in CSS is used to set the color of HTML elements. Typically, this property is used to set the background color or the font color of an element.
  • In CSS we use color values for specifying the color. We can also use this property for the border-color and other decorative effects.
  • We can define the color of an element by using the following ways:
  1. Built-in color.
  2. Hexadecimal notation.
  3. RGB format.
  4. RGBA format.
  5. HSL.
  6. HSLA.
  • Built-in Color:
As its name implies, built-in color means the collection of previously defined colors that are used by using a name such as red,blue,green,etc.

        Syntax:
                Color:Color-name;
  • Hexadecimal notation:
Hexadecimal can be defined as a six-digit color representation.This notation starts with the # symbol followed by six characters ranges from 0 to f. In this, the first two digits represent the red(RR) color value, the next two digits represent the green (GG) color value, and the last two digits represent the blue(BB) color value.
The black color notation in hexadecimal is #000000,and the white color notation in hexadecimal is #FFFFFF.

  • RGB Format:
RGB Format is the short form of 'RED GREEN and BLUE' that is used for defining the color of an HTML element simply by specifying the values of R,G,B that are in range of 0 to 255.
The color values in this format are specified by using the rgb() property. this property allows three values that can either be in percentage or integer(range from 0 to 255)

        Syntax:
                color:rgb(R,G,B):

  • RGBA Format:
It is almost similar ta RGB format except that RGBA contains A (Alpha) that specifies that elements transparency.The value of alpha is in the range 0.0 to 1.0, in which 0.0 is for fully transparent.

        Syntax:
               color:rgba(R,G,B,A):

  • HSL:
It is a short form of Hue, saturation, and Lightness. Let's understand them individually.

Hue: it can be defined as the degree on the color wheel from 0 to 360. 0 represents red, 120 represents green, 240 represent blue.
Saturation: it takes value in percentage in which 100% represents fully saturated, i.e., no shades of gray.
Lightness: The lightness of the color can be defined as the light that we want to provide the color in which 0% represents black (there is no light),50% represents neither dark nor light, and 100% represents white (full ligtness).

        Syntax:
                 color:hsl(H,S,L);

  • HSLA:
It is entirely similar to HSL property, except that it contains A (alpha) that specifies the elements transparency. The value of alpha is in the range 0.0 to 1.0 in which 0.0 indicates fully transparent, and 1.0 indicates, not transparent.

        Syntax:
                color:hsla(H,S,L,A);

CSS Border:

  • The CSS border propeties are use to specify the style, color and size of the border of an element.
  • The CSS Border properties are given below
  1. border-style
  2. border-color
  3. border-width
  4. border-radius

  • CSS border-style:

The Border style property is used to specifiy the border type which ypu want to display on the web page.
There are different types CSS border-styles:

Border-style:
none dotted dashed solid double groove ridge inset outset

We can apply the border styles individually:
Border-top-style Border-right-style Border-bottom-style Border-left-style


  • CSS border-color

There are different methods to set the color of the border.
  1. Name: it specifies the color name. for example: "red".
  2. RGB: it specifies the Rgb value of the color. for example: "rgb(255,0,0)"
  3. Hex: it specifies the hex value of the color. for example: #ff0000
  4. HSL: it specifies the RGB value of the color. for example: "hsl(300,50%,50%)"

  • CSS border-width:

The border-width property is used to set the border's width. it is set in pixels. you can also use the one of the three predefined values, thin, medium or thick to set the width of the border.

  • CSS border-radius:

This CSS property sets the rounded borders and provides the rounded corners around an element, tags, or div. it defines the radius of the corners of an element.

CSS-Margins:

  • CSS Margin property is used to define the space around elements. it is completely transparent and doesn't have any background color. it clears an area around the element.
  • Top, bottom, left and right margin can be changed independently using separate properties. you can also change all properties at once by using shorthand margin property.
  1. The margin-bottom specifies the bottom margin of an element.
  2. The margin-top specifies the top margin of an element.
  3. The margin-left specifies the left margin of an element.
  4. The margin-right specifies the right margin of an element.

  • Margin: Shorthand Property:

CSS shorthand property is used to shorten the code. it specifies all the margin properties in one property.
There are four types to specify the margin property. you can use one of them.
  1. margin: 50px 100px 150px 200px;
  2. margin: 55px 100px 150px;
  3. margin: 50px 100px;
  4. margin: 50px;

CSS padding:

  • CSS padding property is used to define the space between the element content and the element border.
  • It is different from CSS margin in the way that CSS margin defines the space around element. CSS padding is affected by the background colors. it clears an area around the content.
  • Top, bottom, left, and right padding can be changed independently using separate properties. you can also change all properties at once by using shorthand padding property.

Float Property:

  • The float property in CSS is used to specify the horizontal alignment of an element within its parent container. When an element is floated, it is removed from the normal flow of the document and shifted to the left or right of its parent container. The elements that follow the floated element will flow around it, unless otherwise specified.

  • The float property can be set to one of the following values:
  1. left: The element will be floated to the left of its parent container.
  2. right: The element will be floated to the right of its parent container.
  3. none: The element will not float and will remain in the normal flow of the document.
  4. inherit: The value of the float property will be inherited from the parent element.
