HTML lists

HTML lists are used to sepcify lists of information. all lists may contain one or more list elements.
there are three different types of html lists

Ordered list or numbered list(ol)
unordered list or bulleted list (ul)
description list or definition list (dl)



HTML ordered list :

In the ordered html lists, all the list items are marked with number by default.it is known as numbered list also.
the ordered list starts with ol tag and the list items start with li tag.

use type attributes to change default num to roman numbers, alphabets and middle numbers also.
 put into ol starting tag inside only.

ex:(type = "I" or "i" or "A" or "a") , (start = "5"), (type = "A" start = "7") .

Html unordered list or bulleted list :

In html unordered list, all the list items are marked with bullets. it is also known as bulleted list also.
unordered list starts with ul tag and list items start with the li tag.
 put into ul starting tag inside only.

use type attributes to change default bullets to  disc or ciricle or square 

ex :  type = disc or square or circle
