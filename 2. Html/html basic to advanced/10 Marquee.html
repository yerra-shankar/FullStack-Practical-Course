Marquee tag in html

The marquee html tag is a non standard html element which is used to scroll a image or text horizontally or vertically.

In simple words, you can say that it scrolls the image or text up, down, left or right automatically,and use width or height also

use scrollamount attribute to increase the speed of img or text.ex: scrollamount = 100. put into marquee starting tag inside only.
use bgcolor  to change the background color of text, and use width also, it scroll based on our percent(%)
      ex: bgcolor = "red" width = 50%.put into marquee starting tag inside only.
use direction attributes to scroll the image and text to right,left,down,up.
usebehavior to change the text or image behavior
ex: direction = right or left or down or up. put into marquee starting tag inside only.
