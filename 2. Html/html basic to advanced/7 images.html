images in html

synatx : img src=""alt""/      ()

html img tag is used to display image on the web page.html img tag is an empty tag that contains attributes only,closing tags are not used in html image element.

the src and alt are important attributes of html img tag.

src
it is a necessary attributes that describes the source or path of the image.

alt
the alt attributes define an alternate text for the image, if it can't be displayed.

width
it is an attributes which is used to specifiy the width to display the image.

height
it is used to specify the height to display the image.

align
it is changing position of photos or texts(left to right)
