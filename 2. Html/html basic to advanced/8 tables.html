Tables in html

the html tables allow web authors to arrange data like text, images, links, other tables, etc.
into rows and columns of cells.

The html tables are created using the table tag in which the tr tag is used to create
tables rows and td tag is used to create data cells .the elements under td are regularand left aligned by default.
th is used to tableheading.


border is used to increase the tableborder. ex: border = 2 .put into table starting tag inside only.
cellpadding is used to increase the cells sizes. ex: cellpadding= 5. put into table starting tag inside only.
cellspacing is used to ircrease the border space. ex: cellspacing= 8. put into table starting tag inside only.
bordercolor is used to changing the bordercolor of table. ex: bordercolor= "red". put into table starting tag inside only.
bgcolor is used to changing the bgcolor of table. ex: bgcolor="yellow" . put into table staring tag inside only.
height,width is used to increase the table. ex:height=400,width = 400. put into table staring tag inside only.
caption is used to caption of tables
rowspan is changing into one name into num of rows. ex: rowspan = 2. put into td starting tag inside only.
colspan is changing into one name into num of columns.ex: colspan = 3. put into td starting tag inside only.
